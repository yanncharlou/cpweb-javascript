Pratique si on veut utiliser une chose différente entre plusieurs valeurs possibles d'une variable.

```js
switch (prenom) {
    case "didier":
        //code à executer
        break;
    case "julien":
        //code à executer
        break;
    default:
        //code à executer par défaut
}
```
Ne pas oublier le `break` à la fin de chaque condition.
